# ALL SCAN

ALL SCAN is a professional Nmap-based network scanning tool for **authorized security testing** on Kali Linux.

---

## Features
- Single IP & IP range scanning
- Service and version detection
- Default NSE scripts
- OS detection
- Open ports summary
- Insecure service detection
- Automatic report generation
- No sudo required to run

---

## Installation

```bash
git clone https://github.com/Kripal-pv/all-scan.git
cd all-scan
sudo bash install.sh
